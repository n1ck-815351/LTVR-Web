LTVR Administrator Link Manager

This link manager's purpose is to generate links for organization owners. Generally, you should only need to generate 1 administrator account for them, as they can then generate the links they need for their staffing situation.
The system has different types of users, and each user type has different levels of permissions:

  • Administrators will be able to manage schools, and any accounts lower than their own.
  • Moderators will only be able to manage accounts lower than their own.
  • Teachers will be able to create, edit, and delete classes within their assigned schools/orgs, and be able to create student "accounts" for use in VR.
  • Students will be able to view their schedule and information about their school and org.

Users can be assigned multiple roles. As an example, you can assign a user as a Moderator and a Teacher so that they can also manage classes.
